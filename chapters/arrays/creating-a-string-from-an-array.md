## Problem

You want to create a string from an array.

## Solution

Use JavaScript's Array toString() method:

{% highlight coffeescript %}
["one", "two", "three"].toString()
# => 'three,two,one'
{% endhighlight %}

## Discussion

`toString()` is a standard JavaScript method. Don't forget the parentheses.
